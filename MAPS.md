# Maintain file

Use the PowerShell script below to create the map list.

```powershell
Get-ChildItem | Where-Object { $_.Extension -ne ".map" } | ForEach-Object {
    Write-Host "![$($_.BaseName)](assets/$($_.Name))  "
    Write-Host "$($_.BaseName)  "
    Write-Host ""
}
```
